## 🎯 Giới Thiệu

Website thương mại điện tử chuyên bán các sản phẩm đặc sản Tây Bắc và dược liệu tự nhiên. Được xây dựng với HTML5, CSS3, và JavaScript thuần, tối ưu cho SEO và tốc độ.
